# 🚗 Web de Venta de Autos

Aplicación web desarrollada para la venta de vehículos, permitiendo a los usuarios explorar autos disponibles, consultar información detallada y gestionar posibles compras de manera rápida e intuitiva.

---

## 📌 Descripción

La **Web de Venta de Autos** es una plataforma enfocada en mostrar un catálogo de vehículos disponibles para su compra. Está desarrollada con tecnologías del lado del cliente, brindando una experiencia dinámica, ligera y accesible desde cualquier navegador.

El sistema permite visualizar autos, filtrar opciones y consultar detalles relevantes de cada vehículo.

---

## 🚀 Características

- 🚗 Catálogo de autos disponibles
- 🔍 Búsqueda y filtrado por características
- 🧾 Detalles completos de cada vehículo
- 📋 Interfaz dinámica e interactiva
- 💾 Almacenamiento local (LocalStorage)
- ⚡ Renderizado dinámico con JavaScript

---

## 🛠️ Tecnologías utilizadas

- **JavaScript (ES6+)** → Lógica de la aplicación  
- **HTML5** → Estructura de la web  
- **CSS3** → Estilos y diseño responsivo

## ▶️ Uso

- Navega por el catálogo de autos disponibles
- Filtra vehículos según tus preferencias
- Consulta información detallada de cada auto
---

## 💡 Funcionamiento

La aplicación utiliza JavaScript para renderizar dinámicamente los autos y gestionar la interacción del usuario. Los datos pueden almacenarse localmente mediante LocalStorage, permitiendo persistencia sin necesidad de backend.
